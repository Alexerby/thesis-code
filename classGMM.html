Detailed Description

Fits a two-component Gaussian Mixture Model via the EM algorithm.

The model assumes each observation \( x_i \) is drawn from

\[ p(x_i \mid \theta) = \pi \, \mathcal{N}(x_i \mid \mu_1, \Sigma_1) + (1-\pi) \, \mathcal{N}(x_i \mid \mu_2, \Sigma_2), \]

where component 1 is the anomalous (fast-cancel) cluster and component 2 is the liquidity-consistent cluster. Parameters \( \theta = \{\pi, \mu_1, \Sigma_1, \mu_2, \Sigma_2\} \) are estimated by maximum likelihood via the EM algorithm (Dempster et al. 1977).

Runs the EM algorithm on \( N \) observations of dimension \( D \).

Initialisation — observations are sorted on the first feature; the bottom 20 % are assigned to component 1 (anomalous).

E-step — compute the posterior responsibility of component 1 for each observation:

\[ r_i = \frac{\pi \, \mathcal{N}(x_i \mid \mu_1, \Sigma_1)} {\pi \, \mathcal{N}(x_i \mid \mu_1, \Sigma_1) + (1-\pi) \, \mathcal{N}(x_i \mid \mu_2, \Sigma_2)}. \]

If opts.fixed_lc[i] is true the responsibility is clamped to \( r_i = 0 \).

M-step — update parameters using the soft counts \( N_k = \sum_i r_i^{(k)} \):

\[ \hat\pi = \frac{N_1}{N}, \quad \hat\mu_k = \frac{1}{N_k}\sum_i r_i^{(k)} x_i, \quad \hat\Sigma_k = \frac{1}{N_k}\sum_i r_i^{(k)}(x_i-\hat\mu_k)(x_i-\hat\mu_k)^\top + \lambda I, \]

where \( \lambda \) = opts.reg is a ridge term preventing singularity.

Iteration stops when \( |\ell^{(t)} - \ell^{(t-1)}| < \texttt{tol} \) or max_iter is reached.

Parameters
data\( N \) observations, each a \( n \)-dimensional vector.
optsTuning parameters (iterations, tolerance, regularisation).
Returns
GMMResult with fitted \(\theta\), responsibilities, and \(\hat\pi_{\text{spoof}} = \frac{1}{N}\sum_i r_i\).

Initialises GMM parameters via K-means with a given random seed.

Runs Lloyd's K-means algorithm (K=2) for up to 100 iterations using two randomly chosen data points as starting centroids. The resulting hard cluster assignments are used to compute \(\pi\), \(\mu_k\), and \(\Sigma_k\) for the subsequent EM run.

Parameters

Returns
Initialised GMMParams ready for the EM loop.

Evaluates \(\log \mathcal{N}(x \mid \mu, \Sigma)\) using a precomputed inverse and log-determinant.

\[ \log \mathcal{N}(x \mid \mu, \Sigma) = -\frac{1}{2}\bigl[n\log 2\pi + \log|\Sigma| + (x-\mu)^\top \Sigma^{-1}(x-\mu)\bigr]. \]

\(\Sigma^{-1}\) and \(\log|\Sigma|\) are passed in pre-computed (via LDLT decomposition) so the M-step can reuse them across all \(N\) observations without redundant factorisations.

Evaluates the observed-data log-likelihood \(\ell(\theta) = \sum_i \log p(x_i \mid \theta)\).

\[ \ell(\theta) = \sum_{i=1}^{N} \log\!\Bigl[ \pi \, \mathcal{N}(x_i \mid \mu_1, \Sigma_1) + (1-\pi) \, \mathcal{N}(x_i \mid \mu_2, \Sigma_2) \Bigr]. \]

The log-sum-exp is evaluated directly (no further numerical stabilisation is applied beyond the precomputed LDLT inverses).

std::pair< VectorXd, VectorXd > GMM::Standardize ( std::vector< Eigen::VectorXd > &  data)
static

Z-score standardises data in-place to zero mean and unit variance.

For each feature dimension \( d \):

\[ x_i^{(d)} \leftarrow \frac{x_i^{(d)} - \bar{x}^{(d)}}{\sigma^{(d)}}, \]

where \(\bar x^{(d)}\) and \(\sigma^{(d)}\) are the sample mean and standard deviation over all \(N\) observations. This prevents features with large absolute values from dominating the covariance structure.

Parameters
dataData to standardise (modified in-place).
Returns
\((\bar x,\, \sigma)\) vectors so fitted means can be back-transformed via \(\hat\mu_k^{(d)} \cdot \sigma^{(d)} + \bar x^{(d)}\) if needed.

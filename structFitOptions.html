◆ fixed_reactive

std::vector<bool> FitOptions::fixed_reactive

Optional per-observation constraint. If fixed_reactive[i] == true, r_i is forced to 0 in every E-step (observation is known-reactive). Must be empty or the same length as the data passed to Fit().
